# sample

sample project template

## prerequisites

- JDK 25
- Gradle 9.1.x

## building

on project root folder, run:

```shell
gradle clean build
```

## running

on project root folder, run:

``` shell
java -jar build/*.jar
```

or to build/run at once:

```shell
gradle bootRun
```

## swagger ui

once the application is running, you can access the Swagger UI at: `http://localhost:8080/swagger-ui`
